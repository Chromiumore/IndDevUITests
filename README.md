# Проект автотестов для фронтенда
## Запуск
* Убедиться, что доступен ресурс http://www.uitestingplayground.com/home
* Убедиться, что установлен jdk-21 и maven
* Перейти в корень проекта
* Запустить тест c помощью команды ```mvn test```

## Генерация отчёта
* Перейти в корень проекта
* Запустить сервер с отчётом с помощью команды ```mvn allure:serve```
  и перейти по адресу в терминале
* Либо сгенерировать отчёт в <b>./target/site/allure-maven-plugin</b> с помощью команды
  ```mvn allure:report```
